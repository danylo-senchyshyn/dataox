### Dataox Job Scraper

## This is a simple Spring Boot application designed to parse job vacancies from a website and save them into a PostgreSQL database.

## Main features include automatic extraction of job location, saving unique job postings by URL, and logging of duplicate entries that are skipped.

## The project uses Java 17, Spring Boot, JPA with Hibernate for ORM, PostgreSQL as the database, and Jsoup for HTML parsing.

## To get started quickly, configure your PostgreSQL database connection in the application properties file, build the project using Maven, and run the application. The app runs by default on http://localhost:8080.

## The project is intended as a base for further extension and customization.
